## 安装
```bash
git clone https://github.com/decert-me/nft-collect.git
```
## 编译
```bash
go build
```
## 配置
```bash
# 主程序配置
cp ./config/config.demo.yaml ./config/config.yaml
vi ./config/config.yaml
```
## 运行
```bash
./nft-collect
```
